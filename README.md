# NewsFeed System Design

Sistema de design de um newsfeed semelhante ao Instagram/Facebook, explorando conceitos de arquitetura distribuída, escalabilidade e system design.

---

## Sobre

Este repositório documenta o processo completo de _system design_ para um sistema de _newsfeed_ em larga escala, desde requisitos até decisões de implementação detalhadas.

## Objetivos

- Demonstrar capacidade de projetar sistemas escaláveis
- Documentar decisões técnicas de forma clara
- Explorar trade-offs arquiteturais

## Escopo do Sistema

- Publicação de posts (texto, imagens, vídeos)
- Feed cronológico de posts.
- Interações (curtidas, comentários)
- Conexões (follow/unfollow)
- Notificação (atualizações sobre curtidas, comentários, follows)
